# 🏺 HeritageHaven Treasures

**HeritageHaven Treasures** is a global marketplace concept for buying and selling handmade decorative items inspired by traditional cultures around the world.

From Indian mandala wall art to Moroccan lanterns and Japanese pottery, the platform aims to celebrate craftsmanship and connect artisans with a global audience.

---

## 🚧 Project Status

⚠️ **This project is currently under development and not fully completed.**

Core functionality is implemented, but many features are still being built and improved.

---

## ✨ Current Features

- 🔐 Google Authentication (Firebase)
- 📦 Add Products (Firestore)
- 🗂️ Display Products dynamically
- 🗑️ Delete Products
